GrammarBot Test Page

This page contains different types of text inputs to test the GrammarBot extension.

Grammar Check Example

Right-click on the text and select "GrammarBot > Grammar Check" to test grammar checking.

Chinglish Detection Example

Right-click on the text and select "GrammarBot > Detect Chinglish" to test Chinglish detection.

I very like to use this product. It is good for use. However, I think the price is a little expensive. I already use it for long time. I will introduce this to my friends.

Shorten Example

Right-click on the text and select "GrammarBot > Shorten" to make it more concise.

Expand Example

Right-click on the text and select "GrammarBot > Expand" to add more details.

The new product exceeded sales expectations.

Tone Adjustment Example

Right-click on the text and select "GrammarBot > Change Tone" and choose a tone.
